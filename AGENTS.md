# manshin — 에이전트 공통 작업 규칙 (기준 문서)

한국 무속 호러 덱빌딩 로그라이크. 단일 파일 웹 게임 (`index.html`, 외부 의존성 0).

**이 문서가 모든 AI 에이전트(Codex·Claude·기타)의 공통 기준이다.** 에이전트별 보충 문서(CLAUDE.md 등)는 이 문서와 충돌할 수 없다.

## 실행·검증

- **실행**: `index.html`을 브라우저에서 직접 연다 (macOS 예시: `open index.html`). 서버 불필요, 모바일 세로 360×640 기준.
- **테스트**: `cd tests && ./run.sh` — **ALL_GREEN 아니면 커밋·배포 금지.** Node 18+ 및 bash/sed 필요 (Windows는 WSL 또는 Git Bash).
- `tests/_game.js`는 run.sh가 index.html에서 추출하는 **생성 파일** — 직접 수정하지 않는다 (gitignore됨). 각 스모크의 대상은 `tests/README.md` 참조.
- 수치·상태 기대값은 `node:assert/strict`로 검증한다 (출력만 하는 검증 금지). 렌더 스모크는 무예외 실행을 통과 조건으로 삼을 수 있다. 새 기능·수정에는 대응 assertion을 추가할 것.

## 작업 규칙

- **사용자가 요청하지 않은 커밋·푸시·배포는 수행하지 않는다.**
- 코드·커밋 메시지는 영어, 게임 내 텍스트·주석·문서는 한국어.
- 런 저장 스키마를 바꾸면 `mapVersion`을 올린다 (구버전 저장은 로드 시 폐기됨).

## 게임 규칙 불변식 (적대 밸런스 검토로 확정)

- 恨 상한: 일반 18 / 정예·보스 24 (`gainHan()` 경유 필수)
- 굿거리 턴당 1회 (`gutCastTurn`)
- 낙인은 공격 **카드당** 1회 발동, 철갑 관통, 상한 8
- X소모 카드 최대 4타 / 방어이월 신기 상한 12 / 동일 신령부 중첩 불가
- 힘 획득은 `gainStr()` 경유 (사기 보정 일원화, 강신 임시 힘 제외)
- 출전: 몸주신 + 귀신 `active` 2위. 전투 덱 = `battleDeck()` 필터. 대기 신령도 원한 유지(의도)
- 다단히트 막 공격 보너스는 타수 분배 (`Math.ceil(atkAdd/h)`)
- 소문(`ensureHaewonTarget`)은 `descendantSet` 기반 — 도달 가능한 후손 노드에만

상한 변경은 [docs/CHANGELOG.md](docs/CHANGELOG.md)의 기존 설계 논거를 검토하고, **플레이 데이터·변경 이유·대응 회귀 테스트를 함께 제시**해야 한다.

## 시스템 주의점

- **피해 계산은 `calcIncomingHit()` 한 곳** — 전투 실행과 의도 표시(칩·상세)가 공유한다. 피해 관련 수정은 반드시 이 함수를 거치고, **다단히트는 행동 개시 시 1회 스냅샷** 규칙을 유지할 것.
- **지도 그래프 불변식**: 모든 노드는 시작점에서 도달 가능, 막다른 길 없음 (smoke11이 200회 생성 검증). `genMap` 수정 시 이 테스트를 반드시 돌릴 것.
- **아트 규칙**: 캐릭터는 fillRect 블록 매스만 (arc/ellipse 금지). 색은 무채색 위주 + 강조 원색 1-2곳 — 렌더 파이프라인이 무채색을 4단 흑백램프+디더링으로 바꾼다.
