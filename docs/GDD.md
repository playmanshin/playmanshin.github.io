| 항목 | 내용 |
|------|------|
| 장르 | 무속 호러 덱빌딩 로그라이크 (StS 구조 + 스탠스 시스템) |

## 1. 런 구조 — 네 밤의 진오기굿

**4막(거리)**: 부정거리(서낭당 고개) → 영실거리(목 없는 폐가) → 도령거리(무덤골: 봉분·비석·도깨비불) → 뒷전거리(귀문 어귀).

- **막별 스케일**: 적 HP ×1.0 / 1.22 / 1.5 / 1.85, 공격 +0 / +2 / +4 / +6. 막보스 최종 HP 72 / 103 / 136 / 171
- **막보스 4**: 장산범(내 피해를 되받는 거울 — 의도 상세에 대응법 명시) · 그슨대(매턴 힘+1 증식) · 구미호(흡혈 + 50% 본색: **전용 2페이즈 패턴으로 교체**·힘+2·의도 즉시 갱신·꼬리 전개·핏빛 달) · 지네각시(백족: 사이클마다 다단히트 +1, **50%에서 즉시 +1 급증 1회**)
- **경로 그래프**: StS식 연결 지도 — 노드 ID + `next[]` 간선, 모든 노드 도달 가능·막다른 길 없음(생성 불변식). 解(해원 대상)·변종·판효과가 처음부터 공개되어 몇 수 앞 경로를 계산한다
- **지도 선택 UX**: 탭은 언제나 미리보기(먼 노드 포함, 확정 불가) — 확정은 스티키 바텀시트의 「이 길로 간다」로만(첫 행은 「첫 굿판 시작」). 「다른 길 보기」와 "확정 전에는 바꿀 수 있다 / 나서면 돌아올 수 없다" 안내 고정 표시
- **위험 태그**: 연타·반사·관통·동티·한(恨)·약화·성장·흡혈·철갑·회복·거울 — 실제 사용될 패턴에서 유도해 노드 미리보기에 예고 (동티와 전투의 恨은 별개 태그). 경로 전략(내 덱 상성으로 길 고르기)의 재료
- **4막 상위 패턴**: 뒷전거리에 재등장하는 귀신은 기제 최대 2개를 조합한 `patUp` 사용 — 스탯 재탕이 아니라 수법이 바뀐다. 전부 태그·의도로 사전 고지 (숨은 카운터 금지)
- **난이도 프로필**: **본굿(표준·추천, 기본)** — 당산목 30%·막 종료 15%·천도 20%/최대+2/방어6·명다리 30%/35전·일반전 드래프트 30%·안전 기연 회복 절반. **이야기(관대)** — 기존 수치 보존, 금제 비활성. 난이도는 런 저장에 귀속, 적 능력치는 프로필과 무관
- **런 계측**: 로컬 전용 runStats(막별 HP·피해·회복·평균 턴·경로 분포·보상 선택·승패) — 종료 화면 요약 + JSON 복사. 표준 난이도 목표: 1막 80~90% / 2막 60~75% / 3막 40~55% / 첫 클리어 25~40%, 보스 6~9턴
- **자동 저장**: 지도가 보일 때마다 런 전체 저장(localStorage), 타이틀에서 이어하기. 승리/패배 시 삭제. 스키마 버전(`mapVersion`) 불일치 시 폐기
- **판(場) 효과** (2막+ 전투 30%): 핏빛 달(쌍방 피해+2) / 그믐밤(의도 종류만 표시, 보상 엽전+8) / 곡소리(신명+1/턴) / 부정 탄 땅(주기적 동티)

## 2. 전투

신력(에너지) 3, 드로우 5, 적 의도 공개. **의도 칩 탭 → 약화·힘 반영 실제 예상 피해와 기제 설명** — 의도 표시와 실제 피해는 단일 계산 함수를 공유하며, 다단히트는 행동 개시 시 1회 스냅샷으로 파리티를 보장한다.

**상태·기제**: 恨(턴 기반 도트, 상한 일반 18/정예·보스 24) · 낙인(히트 기반 도트 — 공격 카드당 1회, 철갑 관통, 상한 8, 소멸 없음) · 약화 · 힘 · 방어 · 철갑(공격 감쇄, 恨·굿거리 관통, 상한 4) · 공포(신명 드레인) · 관통(방어 무시) · 반사(타격당 발동).

**적의 덱 공격**: 저주 의도로 내 덱에 동티를 꽂는 적, 자가회복 적.

## 3. 강신(降神) — 스탠스 시스템

출전 신령 중 **하나에 빙의**한 상태로 싸운다. 신력 1로 전투 중 교체 — 신기(오라)가 통째로 바뀐다. 턴 안의 스탠스 댄스가 콤보를 만든다: 두억시니로 힘을 쌓고 → 처녀귀신으로 恨을 얹고 → 저승사자의 「명줄끊기」로 수확. **봉인 선택이 곧 빌드 선택.**

**굿거리(궁극기)**: 카드 5장 = 신명 5 → 빙의 신령의 대목 발동, **턴당 1회**. 전 신령 고유 (참수 / 초혼: 죽음 무효 / 씨름 한판: 적 의도 넘어뜨림 / 명부 집행: 비례 피해 / 독 옮기기: 恨 전이 …). 발동 시 컷인 + 지전 흩날림.

## 4. 출전 체계 — 신단과 굿판

**출전 3위**: 몸주신(상시) + 봉인 귀신 2위. **출전 신령의 카드만 전투 덱에 들어가고**(+범용 부적), 강신·굿거리도 출전만 가능. 편성은 당산목·장터·막간의 "신단 정비"에서만.

**대기 신령의 가치**: 원한은 대기 중에도 유지(의도). 신장대(신단 규모 비례)·해원굿·사혼굿·위령제는 신단 전체 대상 — 전투력은 출전이, 서사·경제는 신단이 담당.

## 4-1. 튜토리얼·용어 체계

- **상황별 팁 8종**(ID별 1회 노출, 도움말에서 초기화): map_preview / card_play / intent / reward_choice / first_seal / lineup_channel / wonhan_threshold / gut_ready
- **이중 표기 원칙**: 세계관 용어는 명사로 남기고 버튼은 결과가 보이는 동사로 — "신단에 거둔다 — 봉인", "보내주고 회복한다 — 천도", "신령을 강화한다 — 치성", "필살굿 — 굿거리"
- **원한/恨 분리(P0)**: 신단 원한(런 자원, 4마다 동티 1장)과 전투의 한(恨, 매 턴 피해 후 1 감소)은 별개 — HUD 원한 칩 탭 설명 + 원한 3 도달 시 임계 예고
- **보상 미리보기**: 선택 전에 실제 변화량 표시 — 봉인(원한 W→W'·동티 C→C'장), 천도(체력 H→H'·최대체력·명복 상세)
- **타이틀**: 이어하기(최상단 primary) → 새 굿(2단 덮어쓰기 확인) → 처음 배우기(팁 초기화+시작) → 도움말·용어(시스템별 사전) → 설정(소리·번쩍임·난이도) → 금제

## 5. 봉인 vs 천도 — 탐욕의 딜레마

전투 승리 후 쓰러진 귀신 앞에서 선택한다. 귀신마다 **사연** 한 줄이 붙는다.

| 선택 | 얻는 것 | 대가 |
|------|--------|------|
| **봉인** | 신기(오라) + 굿거리 + 카드 3장 풀킷 | **원한 1~3** (출전 가득 차면 자동 대기) |
| **천도** | 회복 30% + 최대체력 +3 + **명복**(다음 전투 시작 방어8·힘1) | 힘은 남지 않는다 |
| **치성** | 신령 하나 강화 (카드 일괄 업그레이드) | — |

**원한 경제**: 신단 원한 총량 4마다 매 전투 덱에 **동티(저주) 1장**. 위령제(원한 −2)·천도재(방생)로 정화. 원한 12 초과 시 **역빙의** — 가장 사나운 봉인 귀신과의 강제전, 패배 시 전용 배드엔딩.

## 6. 업(業)과 엔딩

봉인 = **신위(神威)**, 천도·위령제 = **공덕(功德)**. 업에 따라 무복 색이 변하고(밤빛↔새벽빛) 이벤트 선택지가 갈린다. **엔딩 4**: 흑무(귀문의 새 주인) / 성무(큰무당) / 만신 / 역빙의 배드엔딩.

## 7. 해원(解冤) — 싸우지 않는 길

신물 6종(족두리·꽃신·제삿밥·청실홍실·마른 옷·엿가락)을 구해 **사연이 맞는 원혼**에게 건네면 전투 없이 해원 봉인(원한 0) 또는 해원 천도(대회복). 원혼 6종(처녀·야광·객귀·손각시·물귀신·염매)만 가능 — 요괴 계열은 의도적으로 불가(민속적 구분). 신물 획득처: 혼수함·엿장수·빨래터·밥짓기·장터(45전). 신물을 얻으면 도달 가능한 앞길에 대상 귀신이 배치되고 지도에 解로 표시된다. **사혼굿**: 처녀귀신+몽달귀신을 함께 들이면 영혼 혼례 — 둘 다 해원 + 공덕 +4.

## 9. 막간 강화 (몸주신 정체성)

막 종료마다 몸주신 전용 강화 2택 × 3회 = 8경로. 현재 장군신(선봉/군진 → 연격/사기 → 참수/불퇴)·바리공주(길잡이/명줄 → 꽃값/저승길 → 원념/헌화) 구현, **나머지 4 몸주신은 로드맵**.

## 10. 콘텐츠 규모

몸주신 6 (장군신·용신·산신·칠성신·바리공주·삼신할미) · 봉인 귀신 18 (원혼/요괴 계열) · 막보스 4 · 변종 3 (굶주린: 흡혈 / 성난: 공격+2 / 묵은: HP×1.35) · 카드 90+ · 무구 10 · 판효과 4 · 엔딩 4 · 금제(반복 난이도) 0~3.

**경제**: 엽전(전투 보상) → 장터에서 부적·무구·명다리·수리취떡·소각·신물 구매. 무구 3택1 드래프트는 정예 100% / 일반 50%.

## 11. 비주얼·사운드

**오방색 × 밤 굿판.** 남색 밤하늘과 달(보스 페이즈에 핏빛), 신목의 오색 천, 촛불, 필름 그레인. 카드는 부적 형태(노란 종이 + 붉은 획). 강신하면 빙의 신령의 형상이 무당 뒤에 어른거린다. 캐릭터 28종은 전부 fillRect 블록 매스 → 4단 램프·베이어 디더·외곽선 파이프라인을 거친 절차적 픽셀 스프라이트, 12fps 스텝 애니메이션. 아이콘은 한자(鬼凶緣木市門·薦祈·眠慰焚飯)로 OS 이모지 편차를 제거.

**전투 연출은 1세대 문법.** 대각 구도(적 우상 ↔ 적 정보 좌상 / 무당 좌하 ↔ 무당 정보 우하)로 창이 캐릭터를 덮지 않는다. 타격은 히트스톱 + 피격 순백 실루엣 + 화면 플래시(설정: 전체/약하게/끔, 다단히트는 마지막 타만). 카드 연출은 fx 6계통 — 참격·부적 투사체(도착 시점 동기화)·恨 안개·烙 낙관·결계 파문·서천꽃. 적 모션은 3유형 — 원혼(부유·찢어지는 피격)/요괴(웅크렸다 도약)/거대귀(화면이 먼저 울림). 죽음은 베이어 디더 디졸브로 재처럼 흩어지고, 恨은 보라 귀기가 수치 비례로 피어오르며 낙인은 烙 문양이 맥동한다.

**사운드는 전부 WebAudio 절차 생성** (외부 오디오 파일 0). 카드 계통별 시전→적중→잔향 3레이어, 연타는 첫 타 강조·중간 축약·마지막 재강조. 전투 BGM은 저역 드론 + 밤바람 노이즈 + 성근 계면조 종소리로, 타격·굿거리 때 3~5dB 덕킹된다. 저체력 시 심장박동.

## 12. 수익화 방향 (초안)

무속 소재 = 코스메틱 광맥: 무복 스킨, 신단 장식, 귀신 도감(설화 해금), 시즌 굿(테마 귀신 추가). 가챠·P2W 금지 원칙. K-호러/K-무속의 글로벌 어필(《파묘》, 《악귀》 계열)이 마케팅 축.
