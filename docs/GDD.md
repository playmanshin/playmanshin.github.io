## 1. 핵심 시스템 세 개

### 1.1 봉인 vs 천도 — 탐욕의 딜레마 (덱몬의 '포획' 개선판)

전투 승리 후 쓰러진 귀신 앞에서 선택한다. 귀신마다 **사연**이 한 줄 붙는다 ("혼례 전날 우물에 빠졌다. 아직 족두리를 기다린다").

| 선택 | 얻는 것 | 대가 |
|------|--------|------|
| **봉인** | 그 귀신의 카드 3장 + 신기(오라) + 강신 스탠스 | **원한 1~3이 쌓인다** |
| **천도** | 회복 40% + 최대체력 +3 | 힘은 남지 않는다 |
| **치성** | 신령 하나 강화 (카드 일괄 업그레이드) | — |

**원한 경제**: 신단의 원한 총량 4마다 매 전투 덱에 **동티(저주 카드) 1장**이 섞인다. 많이 부릴수록 강해지고, 덱은 불길해진다. 포켓몬식 "다 잡기"의 유혹을 저주로 과금하는 구조. 모닥불(성황당)에서 **위령제**(원한 -2)나 **천도재**(귀신 방생)로 정화할 수 있다.

### 1.2 강신(降神) — 스탠스 시스템 (게임성 개선의 핵심)

신단의 신령·귀신 중 **하나에 빙의**한 상태로 싸운다. 전투 중 **에너지 1로 언제든 갈아탈 수 있다.**

| 신령 | 신기 (강신 중 효과) |
|------|-------------------|
| 장군신 | 힘 +2 |
| 용신 | 턴 종료 시 방어 +4 |
| 산신(백호) | 턴 시작 시 회복 2 |
| 처녀귀신 | 공격 카드가 한(恨) 1을 얹는다 |
| 도깨비 | 공격이 적 방어를 무시한다 |
| 어둑시니 | 방어도가 턴을 넘겨 이월된다 |
| 두억시니 | 공격 카드마다 힘 +1 |
| 저승사자 | 턴 시작 시 적 5 피해 |
| … | 귀신 9종 + 몸주신 3종, 전부 고유 오라 |

턴 안에서의 스탠스 댄스가 콤보를 만든다: 두억시니로 힘을 쌓고 → 처녀귀신으로 한을 얹고 → 저승사자의 「명줄끊기」(한 1당 피해 +1)로 수확. **봉인 선택이 곧 빌드 선택**이 된다.

### 1.3 한(恨) — 테마 있는 지속피해

도트 피해의 이름이 '한'이다. 처녀귀신 빌드는 한을 쌓고, 「한풀이」는 적의 한을 2배로 만든다. 적도 저주 의도(🃏)로 **내 덱에 동티를 꽂아넣는다** — 덱을 공격하는 적.

## 2. 전투 (StS-lite 유지 + 개선)

에너지 3, 드로우 5, 적 의도 공개. 상태: 한 / 약화 / 힘 / 방어. 적 의도에 저주(덱 오염)와 자가회복 추가. 보스 구미호는 체력 50%에서 **본색을 드러낸다** (힘 +3 페이즈 전환).

## 3. 등장 귀신 (한국 설화 기반)

야생: 처녀귀신 · 달걀귀신 · 도깨비 · 물귀신 · 어둑시니 · 야광귀 · 객귀
정예: 두억시니 · 저승사자 / 보스: **구미호** (아홉 꼬리 연출)
몸주신(스타터 3택1): 장군신 · 용신 · 산신(백호)

## 4. 비주얼 방향

**오방색 × 밤 굿판.** 남색 밤하늘과 달, 신목의 오색 천, 촛불, 멍석 위의 애기무당(색동 소매·방울·부채). 카드는 **부적** 형태(노란 종이 + 붉은 획). 강신하면 빙의한 신령의 형상이 무당 뒤에 크게 어른거린다.

## 5. 프로토타입이 검증하는 것

1. 강신 스탠스 전환에 에너지를 쓰는 게 재미있는 결정인가 (StS 대비 +α)
2. 봉인/천도가 사연 텍스트 한 줄로 감정적 무게를 갖는가
3. 원한→동티 경제가 "다 잡기"를 실제로 억제하는가

## 6. 수익화 방향 (초안)

무속 소재 = 코스메틱 광맥: 무복 스킨, 신단 장식, 귀신 도감(설화 해금), 시즌 굿(테마 귀신 추가). 가챠·P2W 금지 원칙 유지. K-호러/K-무속의 글로벌 어필(《파묘》, 《악귀》 흥행 계열)이 마케팅 축.
